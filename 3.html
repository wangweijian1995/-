简介¶

分析的数据集来源为“电影数据库”（TMDb，The Movie Database），包含一万余条电影信息，主要信息包括受欢迎程度、收入及预算、电影原标题、演员及导演、发行日期、电影描述关键词等。探索的问题为：每年最受欢迎的电影的体裁是哪些？查询的关键信息为年份、受欢迎程度及体裁，查看每年最受欢迎电影的体裁，并分别统计每种体裁的数量；票房高的电影有哪些特点？关键信息为票房、受欢迎程度、评分数量、预算等。

数据清理：删掉无关列、电影体裁含有 NaN 值的行、重复行¶

电影主页（homepage）、品牌理念（tagline）制片公司（production_companies）缺失值比较多，对于分析问题也几乎无帮助，所以删掉这三列。imdb_id 与分析无特别关系，删掉该列，有体裁（genres）数据，则关键词（keywords）概述（overview）列对分析也无很大影响，删掉。对于票房及预算来讲，计算通货膨胀数据对票房大小的对比无关，删掉。

In [250]:
df.drop(['homepage', 'tagline', 'production_companies', 'imdb_id','keywords', 
         'overview', 'budget_adj', 'revenue_adj'], axis=1, inplace=True)  # 删掉无关列
df['bool'] = df['genres'].isnull()
df.drop(df[df['bool'] == True].index, inplace=True)  # 删掉电影体裁含有 NaN 值的行
df.drop(columns='bool', inplace=True)

删掉电影体裁含有 NaN 值时，通过添加新的 bool 列，判断此行是否有空值，True 为含有空值，所以删掉该行 bool 列值为 True 的行。然后删掉 bool 列，其他行的空值如演员、导演对问题分析无特别关系，可以不删掉带有空值的行，以防删掉一些电影数据。

结论¶

每年最受欢迎的电影的体裁是什么？¶

图表数据显示，大部分年份最受欢迎的电影的体裁按照排名先后为冒险、动作、科幻、惊险类电影。极少数年份最受欢迎的电影为音乐、浪漫体裁。数据包含了 1960 年到 2015 年所有每年最受欢迎的电影，大致能够体现出电影受欢迎程度，但不能保证电影只要包含前几名的体裁就一定会受欢迎。因为去除异常值和空值时，做了一些数据的测试，测试结果对数据影响不大，为了保证报告简洁最终删掉了测试代码。

票房高的电影有哪些特点？¶

图表数据显示，票房高的电影的受欢迎程度、预算、投票数量相对较高，与电影时长、投票平均分、发行年份则无很大关系。其中票房较低的大部分电影，受欢迎程度的值都在票房较高电影的中位数以下；大部分票房较低电影的预算不到票房较高电影预算中位数的一半；而票房较高和较低的电影评分数值虽然相差不大，但是票房较高的电影评分值更加集中，票房低的电影评分值相对分散。当然，这些数据对单个电影来讲可能并不成立，比如，有预算非常高的票房低的电影。
