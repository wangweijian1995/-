简介¶

这个项目是关于电子商务网站的 A/B 测试的结果及分析，弄清楚他们是否应该使用新的页面，保留旧的页面，或者应该将测试时间延长，之后再做出决定。

II - A/B 测试¶

下面采用中心极限定理模拟零假设¶

假定旧的页面效果更好，零假设和备择假设是:

$$H0:p_{new} - p_{old} <= 0$$ $$H1:p_{new} - p_{old} > 0$$

用样本进行五千次抽样，分别求每次的新旧转化率的差，样本数量级降低了些电脑才跑得动。。。

In [54]:
diffs = []
for _ in range(2000):
    sample = df2.sample(df2.shape[0], replace=True)
    p_new = sample.query('group == "treatment"').converted.mean()
    p_old = sample.query('group == "control"').converted.mean()
    diffs.append(p_new - p_old)
In [55]:
plt.hist(diffs);

2000 个样本的转化率（也就是均值）差的分布是正态分布，该样本及更极端情况出现的概率为 p 值

In [56]:

内置程序 （built-in）来实现类似的结果

In [62]:
import statsmodels.api as sm

convert_old = df2.query('group == "control"').converted.sum()
convert_new = df2.query('group == "treatment"').converted.sum()
n_old = df2.query('group == "control"').shape[0]
n_new = df2.query('group == "treatment"').shape[0]

z 分数和 p 值

In [63]:
z_score, p_val = sm.stats.proportions_ztest(
    [convert_old, convert_new], [n_old, n_new], alternative='smaller')
z_score, p_val
Out[63]:
(1.3109241984234394, 0.9050583127590245)

结论一样，p 值大于 0.05，所以不能认为新旧页面有区别。

总结¶

新旧页面并无显著差异，可能需要继续监控，或者隔一段时间监控一次，此次没有考虑时间因素，可能和时间因素有关。
